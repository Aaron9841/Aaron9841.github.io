Intelligent Manufacturing Innovation Laboratory 智慧製造創新實驗室
(暱稱: No. 25 Lab)
這裡是#25號實驗室，以智慧製造場域相關應用研究為主，正直、創新、無極限是實驗室的精神，期望每一位成員都能夠朝此目標邁進。正直就是腳踏實地、不剽竊別人的研究；創新可以是多種技術結合的創意或是全新的發明，包括實體或是新穎的觀念；無極限就是不要被自己、被環境所限制，總是保持不怕失敗的態度。
以下的主題僅供參考，您可以去探索一下，看是不是有興趣做。如果您有自己想做的題目，但不在這個範疇內，或許可以找我討論一下，看是不是實驗室能有資源可以支持您的研究。 自主思考能力的培養，探索未知的領域，應該是比做實驗、寫論文、完成計畫重要。
#1 研究主題 :
5G網路技術 | 5G Network Technology
Research on 5G Network Deployment Technology and Information Security Issues in Smart Manufacturing Fields
計畫概要 :
5G企業專網是未來智慧製造的主要通訊網路，5G RAN (radio access network)及5G Core需要進行設計，才能夠達成 (eMBB, enhanced mobile broadband), 極度可靠低延遲(URLLC, ultra reliable low latency communications)、大量機器對機器室內定位(mMTC (massive Machine Type Communications)。本研究主要針對5G網路部署、應用的規劃，以及實際訊號的測量技術進行分析。目標為提供國內製造業5G網路建構技術資訊。
關鍵字 :
GNS3, mininet, Winprop, SDN (opendaylight), NFV(Opnfv), Network, MEC. (Virtualbox)
#2 研究主題 :
室內定位 | Indoor positioning
Application of Bluetooth Low Energy Indoor Location System in Factory
計畫概要 :
本計畫的室內定位訊採用BLE訊號源，目標為達成公分(cm)等級的定位，運用在智慧製造場域的資材、設備、移載裝置的位置追蹤。
關鍵字 :
BLE beacon, Python
